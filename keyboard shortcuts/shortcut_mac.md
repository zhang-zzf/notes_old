通用：

```
command + tab/command + `	切换窗口
command + h						隐藏窗口
command + m						最小化窗口
command + n						new
command + o						open
command + s						save
command + shift + s		save as
command + p						print
command + w						close
command + q						quit

command + control + q	锁屏

Finder 
command + t						新建tab	
command + n 					新建窗口
command + shift + n 	新建文件夹
command + ctrl + n 		以选中的元素新建文件夹（新建智能文件夹）
command + i						显示简介
command + f						find 搜索
```
