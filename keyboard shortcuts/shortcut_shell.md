1. 常用

   ```
   		ctrl+b: 回移一个字符
   		ctrl+f: 前移一个字符
   		ctrl + e 移动光标到行尾
   		ctrl + a 移动光标到行首
   		ctrl + u 删除光标到行首
   		ctrl + k 删除光标到行尾
   		ctrl + h 向前删除一个字符
   		ctrl + d 向后删除一个字符
   		ctrl + w 向前删除一个单词
   		ctrl+r: 搜索历史命令
   		ctrl+p: 上一个命令
   		ctrl+n：下一个命令
   		ctrl+l: 清屏，替代 clear 命令
   		ctrl + z 挂起到后台
   		fg %1 唤醒最后一个挂起到后台的任务
   ```
