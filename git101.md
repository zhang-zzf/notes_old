# git 常用命令

![git repo](pictures/git001.jpeg)

### status 查看工作区状态
> git status
### log 查询提交日志
> git log

> git log -p

### add/commit
  - working directory -> stage		      
    > git add README.md

    > git add .

  - stage -> working directory          
    > git reset HEAD README.md

  - stage -> master                     
    > git commit -m "comment"

  - state/master -> working directory   
    > git checkout -- README.md
